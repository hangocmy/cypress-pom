## 1. HOW TO RUN CYPRESS ON LOCAL

- Install package:
  `npm install`

- Run app:
  `npm start`

- Run cypress with GUI:
  `npx cypress open`
  Choose specific spec file and run

or

- Run all specs without GUI:
  `npx cypress run --e2e --browser chrome`

## 2. HOW TO WATCH TEST RESULT:
